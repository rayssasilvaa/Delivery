# 🍔 Sistema de Delivery (Java)

## 📌 Descrição

Este é um projeto simples de sistema de delivery desenvolvido em Java, com o objetivo de praticar conceitos fundamentais de programação orientada a objetos.

O sistema permite:

* Cadastro de cliente (nome e WhatsApp)
* Escolha do tipo de pedido (Lanche, Pizza ou Açaí)
* Descrição personalizada do pedido
* Exibição de um resumo final

---

## 🚀 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Enum
* Scanner (entrada de dados)

---

## 🧠 Conceitos aplicados

* Classes e Objetos
* Herança
* Encapsulamento
* Enum para controle de opções fixas
* Separação de responsabilidades (Service)

---

## 📂 Estrutura do projeto

```
Pedidos/
│
├── Pessoa.java
├── Pedido.java
├── Menu.java (enum)
├── PedidoService.java
└── Main.java
```

---

## ▶️ Como executar

1. Clone o repositório
2. Abra o projeto em uma IDE (IntelliJ, Eclipse, VS Code)
3. Execute a classe `Main.java`
4. Siga as instruções no console

---

## 💡 Possíveis melhorias

* Adicionar preços aos itens (usando enum)
* Calcular total do pedido
* Permitir múltiplos itens
* Criar um cardápio mais completo
* Integração com banco de dados

# 🍔 Delivery System (Java)

## 📌 Description

This is a simple delivery system project developed in Java, created to practice Object-Oriented Programming concepts.

The system allows:

* Customer registration (name and WhatsApp)
* Selection of order type (Snack, Pizza, or Açaí)
* Custom order description
* Display of a final summary

---

## 🚀 Technologies used

* Java
* Object-Oriented Programming (OOP)
* Enum
* Scanner (user input)

---

## 🧠 Concepts applied

* Classes and Objects
* Inheritance
* Encapsulation
* Enum for fixed options
* Separation of concerns (Service layer)

---

## 📂 Project structure

```
Pedidos/
│
├── Pessoa.java
├── Pedido.java
├── Menu.java (enum)
├── PedidoService.java
└── Main.java
```

---

## ▶️ How to run

1. Clone the repository
2. Open the project in an IDE (IntelliJ, Eclipse, VS Code)
3. Run the `Main.java` class
4. Follow the instructions in the console

---

## 💡 Possible improvements
